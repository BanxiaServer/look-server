Apache Web Server(独立主机用户)

Apache Web Server(虚拟主机用户)

# 将 RewriteEngine 模式打开
RewriteEngine On

# 修改以下语句中的 / 改为您的DOYO系统目录地址，如果程序放在根目录中则无需修改
RewriteBase /

# Rewrite 系统规则请勿修改
